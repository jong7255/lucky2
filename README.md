## 🚀 GitHub 업로드 방법
1. github.com 로그인
2. New repository → 이름 입력 → Create
3. "uploading an existing file" 클릭
4. 위 파일 전체 드래그&드롭
5. Commit changes 클릭

## ▲ Vercel 배포
1. vercel.com → Add New → Project
2. GitHub 저장소 선택 → Import → Deploy
